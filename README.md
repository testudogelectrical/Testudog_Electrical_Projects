This is a Github Repository mainly meant to help organize the projects of Testudog Electrical, and share the KiCAD files between members assigned to the same projects.

As a reminder, you should make your own branch for any new projects you have started.

File naming format should be as follows: [Project Description]_[First Initial][Last Name]_[DD-MM-YY]
Branches should be named after the project.
For example: powerboard_JDuong_06-15-26 should be put under the 'powerboard' branch.

In case there are two projects of the same description, differentiate using your last name and year on the branch..
For example: powerboard_JDuong and powerboard_AMoradi would be two different branches.

DO NOT commit changes to main unless you have been given permission by both PM and Testudog Electrical Leadership. Instead, create branches off of main with a title which succintly describes your project.

This account will be passed down between Testudog Electrical leadership, and should not be monopolized by a single member of leadership.
